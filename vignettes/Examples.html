Examples

There are currently 3 full examples contained withing the inst directory within the package:

Example: add_twitter_card() for Rmd knitted documents

Within inst -> examples, there is an Rmd document called twitter_large.Rmd. When knitted, this document creates a html page containing twitter meta tags that we have stored on www.jumpingrivers.com. The code use to create the meta tags is also in the document. To preview the twitter card for this document, copy and paste the page URL into the twitter card validator.

Example: add_og_card() for Rmd knitted documents

Again, within inst -> examples, there is an Rmd document, this time called open_graph.Rmd. When knitted, this document creates a html page containing open graph meta tags that we have stored on www.jumpingrivers.com. The Rmd document also contains the code to create the meta tags in the header. I have found the best way to preview open graph sharing cards is to download the chrome extension Open Graph Preview.
